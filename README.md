# 实施步骤

1. 创建 github 项目

1. 安装 git-bash

1. 安装 node.js

1. 使用 npm 安装 hexo-cli

1. hexo-cli 初始化项目

    ```
hexo init project-name
    ```
1. 将本地项目与 github 项目合并

    先进入项目目录, 初始化 git 仓库:

    ```
git init
    ```

    将 github 项目拉到本地:

    ```
git pull --allow-unrelated-histories origin master
    ```
